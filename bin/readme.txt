== Description ==

This is a plugin, copy of the original Hello Dolly plugin written in the Idris programming language, using the PHP backend to generate PHP code. When activated you will randomly see a lyric from <cite>Hello, Dolly</cite> in the upper right of your admin screen on every page.
The purpose of this plugin is to demonstrate how plugin development can be also done in other programming languages.
